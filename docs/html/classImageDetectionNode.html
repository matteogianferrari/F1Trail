Detailed Description

Node for marker localization with stereo video frames.

This node estimate the position (3D spatial point) of an ArUco marker relative to the camera reference frame.

cv_bridge::CvImageConstPtr  right_img

Target detection.

Detection is performed with opencv's ArUco marker library. The marker needs to be detected in both images. The projected center of the marker is found for each image and it is used to compute the disparity.

Parameters
[in]left_imgcv_bridge wrap of the received left image message.
[in]right_imgcv_bridge wrap of the received right image message.
