Detailed Description

ROS2 node that performs point cloud clustering using a 2D Lidar scan.

This node converts a 2D Lidar scan into a point cloud, then performs clustering to detect objects.

Applies Euclidean Clustering to group points into distinct clusters.

Segments the point cloud into clusters based on the Euclidean distance between points. Each cluster represents a potential object in the scene.

Converts a 2D Lidar scan into a 3D point cloud.

Each point in the Lidar scan is projected into 3D space by computing its X and Y coordinates using polar coordinates, while the Z coordinate is set to zero.

Parameters
[in]scanLaserScan message containing the Lidar data.
[out]outputPCPointer to the output 3D point cloud.
